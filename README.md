# Purpose
The purpose of this is to run a bare minimal docker container with a go binary but cross compiled on a different platform.

# Requirements
- Docker running
- Go instalation

## Tested only on a macbook
- brew install go
- brew install docker
- docker-machine start dev
- eval "$(docker-machine env dev)"
